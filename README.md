# SDLC (Software Development Life Cycle) for a Task Manager app
 • Name: Daniel Udhawuve
 
 • Matric number: 24/14551
 
 • Department: Computer Science 
# Planning
Define features (e.g., add tasks, deadlines, priorities, reminders). Identify target users (e.g., individuals, teams).
# Analysis
Research existing task managers (e.g., Trello, Todoist). Gather user needs (e.g., simplicity, collaboration).
# Design
Wireframes/UI for task creation, lists, calendar view. Decide tech stack (e.g., React, Node.js).
# Development
Build core features (CRUD tasks, auth, notifications).
# Testing
Unit tests, UI tests. Test collaboration, reminders, sync.
# Deployment
Launch (e.g., web, mobile). Monitor feedback.
# Maintenance
Fix bugs, add features (e.g., integrations, themes).
